### AirUI-VUE 组件库


[在线文档](https://wintelia.github.io)


#### 开发常用命令

```bash
npm run init // 首次安装
npm run dev  // 启动开发模式
npm run dev-css // 主题样式监听，如果开发的组件有使用主题，这个必须开启
npm run build-doc // 编译文档
npm run build-css // 编译主题
npm run build-lib // 编译组件库和主题
npm run publish // 编译组件库和主题，并发布到npm
```
